This webpage is an assignment for the Web Client Development course in Singapore Polytechnic. It also serves to share my experiences and reflections to attended hackathons (may be updated with more out-of-school activities). Note: This page is created when I am a year 1 student in SP (17 year old). Curious on how I do it? Continue scrolling :)

My first time coding with HTML and CSS was way back during my secondary school. Back then, my teacher insisted on using just notepad to code and as a secondary student, I had no idea softwares like dreamweaver exists. Moreover, I remember I was taught of using the traditional and now obsolete table layout for web. Thinking back, it was quite a fun experience learning to code my very first language.
And who would have known... Now I am in poly and the skill actually proved useful.
